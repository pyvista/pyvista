.. _cli_report:

pyvista report
--------------
Command-line interface for generating a PyVista software environment report.

Using ``pyvista report`` is similar to creating a :class:`~pyvista.Report` in Python.

Examples
********
Show a PyVista report.

.. command-output:: pyvista report

API Reference
*************
Show the output from ``pyvista report --help``.

.. command-output:: pyvista report --help
